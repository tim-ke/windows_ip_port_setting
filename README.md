# windows server  IP 遠端桌面port 腳本配置<br>
<br>

## windows_group_ip
1.ip_range填入IP的主機位置，共有8段  (輸入站群IP表)(gateway)<br>
2.填入遠端port<br>
3.填入網卡名稱<br>
4.netmask填入的是 /27<br>
5.填入gateway<br>
6.填入dns<br>
7.用完記得把cmd刪掉<br>
<br><br>

## Remotedesk_set_port.bat
1.修改 set remotedesk_port=要修改的端口<br>
2.執行交腳本即可<br>
